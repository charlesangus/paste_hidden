```python
def _color_int_to_rgb(color_int):
    """Unpack a 0xRRGGBBAA int into (r, g, b) tuple."""
    red = (color_int >> 24) & 0xFF
    green = (color_int >> 16) & 0xFF
    blue = (color_int >> 8) & 0xFF
    return red, green, blue
```

    **Class structure:**

    ```python
    class PrefsDialog(QtWidgets.QDialog):
        def __init__(self, parent=None):
            super().__init__(parent)
            import prefs as prefs_module
            # Seed local working copies — never mutate prefs module vars until accept
            self._local_plugin_enabled = prefs_module.plugin_enabled
            self._local_link_mode = prefs_module.link_classes_paste_mode
            self._local_custom_colors = list(prefs_module.custom_colors)
            self._selected_swatch_index = None  # index into _local_custom_colors
            self._swatch_buttons = []  # parallel list of QPushButton refs
            self._build_ui()
    ```

    **_build_ui layout (QVBoxLayout):**

    1. QCheckBox "Enable paste_hidden plugin" — checked if `self._local_plugin_enabled`. Store as `self._plugin_checkbox`.

    2. QCheckBox "Input nodes paste as links" — checked if `self._local_link_mode == "create_link"`. Store as `self._link_mode_checkbox`.

    3. QFrame horizontal separator (Shape: HLine, Shadow: Sunken).

    4. QLabel "Custom Colors".

    5. A QWidget containing a QGridLayout (`self._swatch_grid_layout`, spacing=2, 8 columns). Store widget ref as `self._swatch_grid_widget`. Call `self._populate_swatch_grid()` to fill it initially.

    6. An HBox (QHBoxLayout) with three QPushButtons: "Add", "Edit", "Remove". All three call `setAutoDefault(False)`. Store as `self._add_button`, `self._edit_button`, `self._remove_button`. Wire signals:
       - Add: `self._on_add_color`
       - Edit: `self._on_edit_color`
       - Remove: `self._on_remove_color`
       Add a QSpacerItem(expanding, minimum) before the buttons so they right-align, or use addStretch() before adding buttons to left-align (use addStretch() at end so buttons are left-aligned — consistent with the layout sketch in CONTEXT.md where [Add][Edit][Remove] appears left-aligned).

    7. QFrame horizontal separator.

    8. QDialogButtonBox with Ok | Cancel. Wire accepted signal to `self._on_accept`, rejected signal to `self.reject`. The QDialogButtonBox manages its own button order and styling — no need to call setAutoDefault on its internal buttons since Enter confirmation in PrefsDialog is handled by the standard QDialogButtonBox behavior (no hint-mode complexity). Store as `self._button_box`.

    **_populate_swatch_grid(self):**
    Clear `self._swatch_buttons`. Iterate `self._local_custom_colors` with enumerate. For each (index, color_int): create a QPushButton (24×24 fixed size, Qt.NoFocus, setAutoDefault(False)). Apply unselected swatch stylesheet using `_color_int_to_rgb(color_int)`. Wire `button.clicked.connect(lambda checked=False, i=index: self._on_swatch_selected(i))`. Add to `self._swatch_grid_layout` at `(index // _SWATCHES_PER_ROW, index % _SWATCHES_PER_ROW)`. Append to `self._swatch_buttons`. After populating, call `self._update_edit_remove_buttons()`.

    **_rebuild_swatch_grid(self):**
    Clear the layout: while `self._swatch_grid_layout.count() > 0`, take item at 0, get widget, call `widget.deleteLater()`. Clear `self._swatch_buttons`. Reset `self._selected_swatch_index = None`. Call `self._populate_swatch_grid()`.

    **_on_swatch_selected(self, index):**
    Set `self._selected_swatch_index = index`. Apply white-border stylesheet to button at index; apply default-border stylesheet to all others. Call `self._update_edit_remove_buttons()`.

    **_update_edit_remove_buttons(self):**
    Enable or disable Edit and Remove based on whether `self._selected_swatch_index is not None`. Use `self._edit_button.setEnabled(...)` and `self._remove_button.setEnabled(...)`.

    **_on_add_color(self):**
    result = nuke.getColor(). If result == 0, return. Append result to `self._local_custom_colors`. Call `self._rebuild_swatch_grid()`. Auto-select the newly appended color: call `self._on_swatch_selected(len(self._local_custom_colors) - 1)`. Also apply the white-border stylesheet to that button.

    **_on_edit_color(self):**
    If `self._selected_swatch_index is None`, return. result = nuke.getColor(). If result == 0, return. Replace `self._local_custom_colors[self._selected_swatch_index]` with result. Preserve `selected_index = self._selected_swatch_index`. Call `self._rebuild_swatch_grid()`. Call `self._on_swatch_selected(selected_index)`.

    **_on_remove_color(self):**
    If `self._selected_swatch_index is None`, return. Remove `self._local_custom_colors[self._selected_swatch_index]`. Call `self._rebuild_swatch_grid()` (which resets `_selected_swatch_index = None` and disables Edit/Remove automatically).

    Swatch stylesheet constants (inline, consistent with ColorPaletteDialog):
    - Selected: `f"background-color: rgb({r},{g},{b}); border: 2px solid white; border-radius: 2px;"`
    - Unselected: `f"background-color: rgb({r},{g},{b}); border: 1px solid #555; border-radius: 2px;"`

    ```python
    def _on_accept(self):
        import prefs as prefs_module
        import menu as menu_module
        # Read current checkbox states into local vars (user may have changed them)
        self._local_plugin_enabled = self._plugin_checkbox.isChecked()
        self._local_link_mode = (
            "create_link" if self._link_mode_checkbox.isChecked() else "passthrough"
        )
        # Flush local working copies to prefs module-level variables
        prefs_module.plugin_enabled = self._local_plugin_enabled
        prefs_module.link_classes_paste_mode = self._local_link_mode
        prefs_module.custom_colors = list(self._local_custom_colors)
        # Persist to disk
        prefs_module.save()
        # Apply plugin_enabled live (menu enable/disable without restart)
        menu_module.set_anchors_menu_enabled(prefs_module.plugin_enabled)
        self.accept()
    ```
