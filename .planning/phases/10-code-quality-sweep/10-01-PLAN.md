- `menu.py F401`: imports of `paste_hidden`, `anchor`, `labels` are used inside Nuke's string-eval menu callbacks (`nuke.menu.addCommand(..., "paste_hidden.copy_hidden()")`). - `tabtabtab.py B008`: QtCore.QModelIndex() as a default argument is an idiomatic Qt pattern in vendored third-party code — do not modify.
- `tabtabtab.py C901`: `nonconsec_find()` complexity 14 is in algorithm-dense vendored third-party code — do not modify.
- `tabtabtab.py SIM`: vendored third-party code — do not modify.
