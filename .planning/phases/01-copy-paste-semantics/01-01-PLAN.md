<objective>
Add the stream-type detection infrastructure: a new constant, a detection helper, a knob-read helper, and anchor creation wiring so that the appropriate link class (PostageStamp vs NoOp) is detected once at anchor creation time and stored on the anchor node.

Purpose: Anchor creation is rare; link pasting is frequent. Putting detection overhead at creation time avoids repeated introspection on every paste. This also fixes the Camera-produces-PostageStamp bug (LINK-03) by ensuring the class is read from the anchor's stored knob rather than re-derived from LINK_CLASSES at paste time.

Output: constants.py gets ANCHOR_LINK_CLASS_KNOB_NAME; link.py gets detect_link_class_for_node() and get_link_class_for_anchor() and an updated get_link_class_for_source(); anchor.py create_anchor_named() stores the detected class on the anchor.
</objective>

2. Add `detect_link_class_for_node(source_node)` function:
   - Purpose: detect the correct link class for a given source node using Nuke API channel inspection
   - Logic:
     - If source_node is None: return 'NoOp' (safe default, LINK-04)
     - If source_node.Class() == 'Dot': return 'Dot'
     - If source_node.Class() is in LINK_CLASSES: return LINK_CLASSES[source_node.Class()]
     - Otherwise attempt channel inspection: call `source_node.channels()` — if the result contains any channel that starts with 'rgba' or 'depth' or 'forward' (common 2D stream channels), return 'PostageStamp'
     - If channels() raises or returns empty: return 'NoOp' (fallback, LINK-04)
     - If no 2D channels detected: return 'NoOp'
   - Wrap the channel inspection in try/except so any Nuke API error returns 'NoOp'

3. Add `get_link_class_for_anchor(anchor_node)` function:
   - Purpose: read the stored link class from the anchor's hidden knob
   - Logic: if ANCHOR_LINK_CLASS_KNOB_NAME in anchor_node.knobs(), return anchor_node[ANCHOR_LINK_CLASS_KNOB_NAME].getValue(); else return 'NoOp' (safe default)

4. Update `get_link_class_for_source(source_node)`:
   - Add a new first branch: if source_node is not None and is_anchor(source_node): return get_link_class_for_anchor(source_node)
   - Keep the existing Dot branch (return 'Dot')
   - Keep the existing LINK_CLASSES fallback as the final else
   - The LINK_CLASSES fallback remains valid for the direct-file-node paste path (no anchor)
