<objective>
Build the foundation for correct link class selection: detect stream type at anchor creation time, store it on the anchor as a hidden knob, and expose a lookup helper so paste can read the class without inspecting the anchor's input.

Purpose: LINK-01/02/03 require PostageStamp for 2D streams and NoOp for Deep/3D streams. The current LINK_CLASSES dict approach breaks for Camera and other non-2D sources because it looks up the *source* node class rather than the *stream* type. Storing the class on the anchor at creation time solves this once; paste just reads it.

Output: New constant `ANCHOR_LINK_CLASS_KNOB_NAME`, new helpers `detect_link_class_for_node()` and `find_anchor_for_file_node()` in link.py, updated `get_link_class_for_source()`, updated `create_anchor_named()` in anchor.py.
</objective>

From link.py — functions to update:
```python
def get_link_class_for_source(source_node):
    """Return the appropriate link node class for a given source node.
    Dot → Dot, LINK_CLASSES lookup, else PostageStamp."""
    if source_node is None:
        return 'PostageStamp'
    if source_node.Class() == 'Dot':
        return 'Dot'
    return LINK_CLASSES.get(source_node.Class(), 'PostageStamp')
```

```python
def create_anchor_named(name, input_node=None):
    """Create an anchor with the given *name* without any user prompt.
    Returns the new anchor node.
    Raises ValueError if *name* sanitizes to an empty string.
    """
    sanitized = sanitize_anchor_name(name)
    if not sanitized:
        raise ValueError(f"Anchor name {name!r} produces an empty sanitized name")
    nukescripts.clear_selection_recursive()
    anchor = nuke.createNode('NoOp')
    anchor.setName(ANCHOR_PREFIX + sanitized)
    anchor['label'].setValue(anchor_display_name(anchor))
    if input_node is not None:
        anchor.setInput(0, input_node)
        anchor.setXYpos(
            input_node.xpos() + input_node.screenWidth() // 2 - anchor.screenWidth() // 2,
            input_node.ypos() + input_node.screenHeight() + 20
        )
    anchor['tile_color'].setValue(find_anchor_color(anchor))
    add_reconnect_anchor_knob(anchor)
    add_rename_anchor_knob(anchor)
    return anchor
```

- Safe pattern: try node.channels(); if result contains 'deep.' prefix → NoOp; else → PostageStamp
